# 8. 地質図学演習（ステレオ投影法による解析）

## ステレオ投影法とは

地質学では、3次元の物質や構造を取り扱いますが、この3次元構造を２次元で表現する方法として、**ステレオ投影法**がよく用いられます。

ステレオ投影法には、**等角投影（ウルフネット）**と**等積投影（シュミットネット）** がありますが、地質図学では地物の位置や方位を取り扱いますので、ウルフネットを使用して構造解析を行います。
